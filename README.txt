This is the Unity working file for the First-Person Detective VR Experience of Wearable Olfaction Device. The folder includes the Unity scene, assets and scripts, along with the built project.

This is a final project of Virtual and Physical Prototyping at Politecnico di Milano, A.Y. 2020/2021.
